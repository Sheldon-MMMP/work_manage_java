# 项目介绍
1. 这是一个基于Java的项目，用于管理个人目标。通过OKR（Objectives and Key Results）框架，帮助用户实现目标管理和结果跟踪。
2. 项目采用Spring Boot框架，数据库采用PostgreSQL。
3. 项目提供RESTful API接口，方便用户进行任务管理和数据交互。

# 功能介绍
1. 登录，注册
2. 目标管理
   - 创建目标
   - 更新目标
   - 删除目标
   - 查询目标列表
   - 归档目标
   - 恢复目标
   - 查询归档目标列表
   - 绑定目标完成奖励
   - 查询目标完成奖励
   - 完成目标
   - 查询已完成目标列表
3. KR管理
   - 创建KR
   - 更新KR
   - 删除KR
   - 查询KR列表
   - 完成KR
   - 查询已完成KR列表
4. 任务管理
   - 创建任务
   - 更新任务
   - 删除任务
   - 查询任务列表
   - 完成任务
   - 查询已完成任务列表
   - 绑定KR
5. 季度总结和年度总结
   - 季度总结
   - 年度总结
6. 日程管理
   - 创建日程
   - 更新日程
   - 删除日程
   - 查询日程列表
   - 绑定任务
   - 拖动任务设置完成任务执行时间
   - 查询任务执行时间
7. 计时器
   - 开始计时器
   - 暂停计时器
   - 重置计时器
   - 查询计时器时间
